Here the few windows that we are using in VAPT and monitoring the system.


| Command          | Description / Syntax                                                                                                |
| ---------------- | ------------------------------------------------------------------------------------------------------------------- |
| `cd` / `chdir`   | Change directory.                                                                                                   |
| `dir`            | List all files/folders in current directory.                                                                        |
| `mkdir` / `md`   | Create a new folder.                                                                                                |
| `rmdir` / `rd`   | Remove an empty directory.                                                                                          |
| `del` / `erase`  | Delete one or more files.                                                                                           |
| `copy`           | Copy file(s): `copy doc1.doc doc2.doc`.                                                                             |
| `xcopy`          | Extended copy (folders + subfolders).                                                                               |
| `robocopy`       | Robust file operations for syncing/mirroring directories                                                            |
| `move`           | Move or rename files.                                                                                               |
| `ren` / `rename` | Rename a file/folder.                                                                                               |
| `attrib`         | Change file attributes.                                                                                             |
| `type`           | Display the contents of a text file.                                                                                |
| `fc`             | Compare two files.                                                                                                  |
| `tree`           | Show directory structure as a tree diagram.                                                                         |
| `subst`          | Assign a drive letter to a folder.                                                                                  |
| `cls`            | Clear the Command Prompt screen.                                                                                    |


| Command      | Description / Syntax                                                                                               |

| `set`        | View or set environment variables (session only).                                                                  |
| `setx`       | Permanently set environment variables in the registry.                                                             |
| `date`       | View or change system date.                                                                                        |
| `time`       | View or set system time.                                                                                           |
| `doskey`     | Enables command history and macro definitions.                                                                     |
| `echo`       | Display messages or turn command-echo on/off                                                                       |
| `prompt`     | Customize the command-prompt display.                                                                              |
| `title`      | Sets the text in the window’s title bar.
